Note: when you are trying to figure out the background-position-x and background-position-y values to isolate a pose from a sprite set, think about the border (e.g. the white border above) as being a fixed window and you are selecting background-position-x and background-position-y values that move the spriteSet image behind the fixed window. So,

  • provide a negative (or lower) background-position-x value to see that part of the bg image that's more to the right.
  • provide a positive (or higher) background-position-y value to see that part of the bg image that's more to the left.
  • provide a negative (or lower) background-position-y value to see that part of the bg image that's farther down.
  • provide a positive (or higher) background-position-y value to see that part of the bg image that's higher up.

Basically you set the numbers backwards from how you'd think to set them !!!
