Creating Simple JS Components (Destructured Parameter Object)

Open up Chrome's Console to check for any JS errors.

This code demonstrates various ways of passing parameters to JavaScript functions:

  • parameter list: like java except you don't have to specify the type of the parameter in the function header.
  • parameter object: the consumer (calling code) does not have to specify all properties, the provider code (function) can specify default values for any properties not specified.
  • destructured parameter object: the consumer code is the same as parameter object (above), but the provider code (function) uses the property names as variables (does not have to extract parameter object properties into variables).
  • destructured parameter object with default values: the consumer code is the same as parameter object (above) but the provider code (function) nicely specifies default values right in the function header (for any parameter object properties that the consumer did not specify).
